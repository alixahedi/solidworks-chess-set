# ♟️ SolidWorks Chess Set

A complete 3D chess set designed using SolidWorks, including all pieces and the chessboard.
This project focuses on parametric design, clean modeling, and realistic assembly.

---

## 📌 Features

* Full chess set (King, Queen, Rook, Bishop, Knight, Pawn)
* Detailed chessboard model
* Parametric design for easy customization
* High-quality CAD modeling using SolidWorks
* Ready for rendering and manufacturing

---

## 🛠️ Tools & Software

* SolidWorks (CAD Design)
* STEP / SLDPRT / SLDASM file formats

---

## 📂 Project Structure

```
/Chess-Set
│── Pieces/
│   ├── King.SLDPRT
│   ├── Queen.SLDPRT
│   ├── Bishop.SLDPRT
│   ├── Knight.SLDPRT
│   ├── Rook.SLDPRT
│   └── Pawn.SLDPRT
│
│── Board/
│   └── ChessBoard.SLDPRT
│
│── Assembly/
│   └── ChessAssembly.SLDASM
│
└── Exports/
    └── STEP Files
```
